# 千禧年半导体晶圆厂 工业生产控制系统
本项目为操作系统课设

## 0.技术栈

本次课设使用 MASM32 x86 Win32 Assembly 构建 GUI 程序，可视化银行家算法、页表置换算法（FIFO、LRU）、进程调度算法，

部分汇编代码难以编写，因此由 g++ -S -O0 模式得到，不确保安全

模拟订单资源分配、订单执行、生产暂存的工业过程。

## 2.系统参数

### 工控机配置

操作系统：Windows 2000

CPU：Pentium III , 550 - 1133 MHz ,单核

内存：128MB

调度周期：1s

### 生产订单

系统支持同时管理 20~50 个生产订单。

### 生产暂存

系统维护容量固定为 16 块的订单暂存区，用于保存等待调度的订单。

订单进入就绪队列时占用 1 块暂存区；订单被调度开始生产后离开暂存区，生产中的订单不占用暂存块。

### 系统功能

系统模拟晶圆厂订单在生产过程中的完整生命周期：

订单创建 → 资源申请 → 进入就绪队列并占用暂存 → 执行生产任务并释放暂存 → 完成订单

在此过程中：

银行家算法负责设备资源分配与安全性检测；

进程调度算法负责生产订单执行顺序管理；

暂存区可视化负责展示等待调度订单对 16 块订单暂存区的占用情况。

### 厂区设备

光刻机、刻蚀机、涂胶机、检测机、墨影机
